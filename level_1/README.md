# Hodor - Level 1

Cheat an online voting contest with an automated script. Includes a series of increasingly difficult challenges. Language of choice: Python 3

<hr>

## Level 1

Has to have a valid user agent. Has to parse the hidden 'key' field's 'value' from a GET request. Has to add the user's 'id' and 'holdthedoor' fields to POST request. Votes successfully 4096 times.
